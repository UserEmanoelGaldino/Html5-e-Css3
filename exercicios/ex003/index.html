EX003 - Emojis

Uma observação importante antes de trabalhar os emojis, é o costume de utilizar várias tags seguidas de quebra de linha para aumentar o espaço entres as tags.
Por padrão, o correto deve usar o CSS para realizar esse ajuste de espaçamento, pois se trata de uma estilização.

Trabalhando simbolos especiais abaixo

® = Marca registrada,
© = Copyright,
™ = Marca registrada em inglês,
€ = Moeda euro,
$ = Moeda dollar,
£ = Moeda libra,
¥ = Moeda Japonesa,
¢ = Centavo Americano,
δ Δ = Delta Maiúsculo e minúsculo,
↑ = ceta para cima,
