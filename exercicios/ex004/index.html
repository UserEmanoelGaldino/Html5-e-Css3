Trabalhando com imagens

Tipos de imagens mais utilizadas na produção de sites

Existem diversos tipos de imagens, porém os formatos que são mais comuns de aparecer no desenvolvimento web são, jpeg e png.

Quando usar cada formato?

Imagens jpeg geralmente são utilizadas para projetos ou momentos que você deseja mais resolução.

Imagens png são mais usados pois permitem animação e transparência.
